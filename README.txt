OneTap Morning — Final Package

Dateien:
- index.html (alles drin, klicksicher)
- manifest.json (PWA-Manifest)
- sw.js (Service Worker, Offline + Cache v3)
- icon-192.png / icon-512.png (App-Icons)

Installation (GitHub Pages):
1) In dein Repo hochladen (Root). Vorhandene Dateien ersetzen.
2) Settings -> Pages -> Branch: main (root).
3) iPhone: Seite in Safari öffnen mit ?v=7 (Cache-Buster), z.B. https://DEINNAME.github.io/onetap-morning/?v=7
4) Teilen -> Zum Home-Bildschirm.

Wenn Updates nicht sichtbar sind:
- Alte Home-Screen-App löschen.
- Safari -> Einstellungen -> Erweitert -> Website-Daten -> github.io löschen.
- Seite erneut mit ?v=7 laden.
